Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Выведутся цифры от 0 до 9, а потом программа уйдет в deadlock

Эта программа создает канал ch для передачи целочисленных значений. 
Затем запускается анонимная горутина, которая отправляет числа 
от 0 до 9 в канал ch. В основной функции программа начинает читать 
значения из канала ch в цикле for range, и для каждого полученного 
значения программа выводит его на стандартный вывод.
Однако, поскольку ни в одной из горутин нет явной инструкции 
закрытия канала, программа будет ждать получения значений 
из канала до бесконечности. После того как анонимная горутина 
отправит все значения в канал, она не завершится, а будет 
блокирована в попытках отправить значение в канал, который 
больше не используется.
Из-за этой блокировки основная горутина также будет заблокирована 
в ожидании получения нового значения из канала, и программа зависнет.
Чтобы избежать этой ситуации, необходимо закрыть канал после 
завершения отправки значений. Это можно сделать с помощью 
функции close(ch), которая будет сигнализировать о том, что 
больше значений не будет отправлено в канал, и основная 
горутина может завершиться после чтения всех доступных 
значений из канала.

```
