Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]

Сначала создается массив a из 5 целочисленных элементов: {76, 77, 78, 79, 80}.
Затем создается срез b, который является частью массива a. Этот срез начинается 
с элемента a[1] (значение 77) и заканчивается элементом a[3] (значение 79). 
При создании среза b мы указываем диапазон [1:4], где включается элемент 
с индексом 1 (значение 77), но не включается элемент с индексом 4 (то есть a[4], 
значение 80).
Затем срез b выводится на экран с помощью fmt.Println(), что приводит к выводу [77 78 79].

```
